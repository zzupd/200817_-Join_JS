[윤년]
1. 1년동안의 달력에 지정된 시간이
    지구 공전 1바퀴와 맞지 않아서 날짜의
    차이를 두기 위한 년(year)

2. 윤년 확인
  1) 어떤 년도를 4로 나누었을 때 나머지가
      0이년 1차적으로 윤년 대상
     (4로 나누었을 때 나머지가 0이 아니면
       모두 평년, 즉 2월이 28일 까지)
  2) 그러나 4로 나누어 떨어지는 년도들 중에서
      100으로 나누어 떨어지면 평년이 된다
  3) 2) 번에 100으로 나누어 떨어지는 평년들중에서
      400으로 나누어 떨어지면 다시 윤년이 된다.

보기1.         나머지  윤년/평년   2월
   1964 =>   0        윤년      29일
   2003 =>   3        평년      28일    
   2020 =>   0        윤년      29일
  즉, 100단위로 나누어 떨어지지 않고
       4로 나누어 떨어지면 윤년, 그렇지 않으면 평년

보기2.         %4   %100  %400   윤년/평년   2월
   1900 =>   0       0        300       평년     28일
   2000 =>   0       0          0         윤년     29일     
   2100 =>   0       0        100       평년     28일
  즉, 100단위로 나누어 떨어지고
       400단위로 나누어 떨어지면 윤년
       100단위로 나누어 떨어지고
       400단위로 나누어 떨어지지 않으면 평년

=> 중첩 if 제어문

    if (해당년도 % 4 == 0) {
      
       중첩if 사용하여
       100으로 나누어 떨어지는 것과
       400으로 나누어 떨어지는 것들을 
       구분하여 윤년과 평년으로 구분함

    } else {
        평년
    }
